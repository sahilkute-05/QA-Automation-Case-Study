# QA Automation Case Study

This repository contains my solution for the **QA Automation Engineering Intern Case Study**.

The solution demonstrates an approach for designing and implementing a scalable automation framework for a B2B SaaS platform using **Playwright**, **pytest**, and **API Testing** concepts.

---

## Project Overview

The case study focuses on three key areas:

- Debugging flaky Playwright automation tests.
- Designing a scalable QA automation framework.
- Implementing an API + UI integration testing approach.

The proposed solution emphasizes maintainability, scalability, cross-platform compatibility, and support for CI/CD pipelines.
